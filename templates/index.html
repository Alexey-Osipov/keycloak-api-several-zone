Автоматизация администрирования нескольких Тестовых Зон Keycloak.

Описание:

  • Создание пользователя

    • Базовое задение ТУЗ с возможностью назначения ролей одного клиента.
    • Назначение ролей уже существующему пользователю.

  • Создание Клиента

    • Создание клиента с возможностью создания ролей (можно передавать списком через запятую: role1,role2,role3).
    • Создание ролей у уже созданного клиента.

  • Загрузить Тему

    • Загрузка архива с Темой на сервера KC и его применние у  указанного клиента.

  • Найти Пользователя

    • Поиск пользователя с выводом назначенных клиентских ролей.

  • Найти Клиента

    • Поиск клиента с выводом заведенных ролей.
